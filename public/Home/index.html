Student Typing Test
A Simple app which let's Teacher view his/her students typing speed in descending order in realtime. The Teacher can create a private room with a secret code, which he/she can then share with the students, so they can join.
The Idea is to show how we can pass data between two website using a common socket connection. This project uses two website Teacher and Student, and a server using SocketIO
